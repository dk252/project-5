List of improvements made

Color Contrast

Changed navigation background color from green to white.

Changed contrasting welcome from white to black with a slightly grey small shadow.

Semantic HTML

Replaced the div class=nav/header with just header/nav elements.

Removed br tags and replaced with p tags.

Replaced font elements with actual heading elements like h1,h2,h3 etc.

CSS

Replaced div nav css elements with just nav to reflect changes in the html document.

Replaced font 7,6,5 css elements with respective h1,h2,h3 elements to properly address the semantic html issues.

Images

Added the alt text tag to images for a textual description of the image for screenreaders to play.

Audio Player

Added a transcipt.html file to transcribe the audio file for hearing impaired users.

Added a direct link to the audio file for browsers that don't support html5 audio.

Form Elements

Added a description of the search input element so it's read out by screenreaders saying to search the site's content.

Show/hide Comment Control Button

Changed the div element that was used for the show/hide comment control to a button element

Table

Added a summary to the table that describes what the table has inside of it.

Changed the first row and column into the th tag for proper table tags.

Specified header cells as rows and columns using the scope attribute with col/row values.

Other Improvements I Made

Added the lang=en-US tag to the html doctype opening.

Changed the color of the opening welcome message above the page to a non contrasting color.
